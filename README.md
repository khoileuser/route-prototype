# route-prototype

Prototype for Displaying Routes and Combine Routes for De Heus Vehicle Booking App

## Config

Create `.env` with `MAPS_TOKEN=<your-google-maps-token>`

## Install

Virtualenv

```bash
python -m venv venv
```

Activate

```bash
venv\scripts\activate
```

Install requirements

```bash
pip install -r requirements
```

Run dev

```bash
fastapi dev main.py
```

Production

```bash
fastapi run
```
